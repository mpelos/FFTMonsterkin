# Live proof — Dragon's Might on a Chocobo

User test, modded game (Beast Tamer + TestKit, clean modloader log):
Dragon's Might cast on a **Chocobo** (MonsterGraphic=1, vanilla dragon-check would
force a miss) **applied its effect**. This proves live, in one shot:

1. OverrideAbilityActionData **Formula override** is honored by the engine
   (first live proof of a Formula re-point, not just X/Y).
2. Formula 0x3B on ability 253 reproduces Dragon's Might behavior on a
   previously-forbidden target => the Dragon Check is fully bypassed.
3. The whole Beast Tamer mechanism (re-point 0x5A-0x5D -> 0x33/0x3C/0x3B/0x12) works.

Remaining spot-checks (expected to pass, same mechanism): Charm hit-%, Gift heal
values + caster HP cost, Speed quick-turn + faith-scaled %.

Released as v1.0.0.
